# Smart Classroom Entry Monitoring System

## Description
An IoT-based classroom entry monitoring system using ESP32 and PIR motion sensor.

## Features
- Motion Detection
- LED and Buzzer Alarm
- Blynk Dashboard Integration
- ThingSpeak Data Logging
- Detection Counter
- Cloud Monitoring

## Hardware
- ESP32
- PIR Motion Sensor
- LED
- Buzzer
- Push Button

## Software
- Arduino IDE
- Wokwi Simulator
- Blynk IoT
- ThingSpeak
